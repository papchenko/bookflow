# BookFlow

**Share Knowledge, Inspire Others**  

BookFlow is a platform for discovering, sharing, selling, and lending books. Instead of letting books gather dust on your shelf, you can lend them as collateral or sell them to other readers. BookFlow is designed for readers who value simplicity, a clean interface, and a seamless user experience.

---

## 🌟 Key Features

- **Book Lending & Sale**: Easily lend your books or sell them to other readers.
- **Discover & Explore**: Browse books shared by the community.
- **Clean UI**: Focused on a minimalistic, intuitive design for easy navigation.
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile.
- **Secure Transactions**: Lending books as collateral with trust and transparency.

---

## 🛠 Technologies Used

- **Frontend**: React.js, Tailwind CSS  
- **State Management**: React Hooks  
- **Routing**: React Router DOM  
- **Forms & Notifications**: React Toastify  
- **Animations & Interactions**: Swiper.js for carousels  
- **Backend / API**: EmailJS for contact forms  
- **Build Tools**: Vite
